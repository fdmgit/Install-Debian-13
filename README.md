# Installation script for Debian 13

To Install AMD64 version (stable) use:

```
wget https://raw.githubusercontent.com/fdmgit/install-debian-13/main/inst_deb13_amd64.sh && bash inst_deb13_amd64.sh
```

To Install AMD64 production version (stable) use:

```
wget -O inst_deb13_amd64 https://raw.githubusercontent.com/fdmgit/install-debian-13/main/inst_deb13_amd64_prod.sh && bash inst_deb13_amd64.sh
```

To Install AMD64 version (UNSTABLE) use:

```
wget https://raw.githubusercontent.com/fdmgit/install-debian-13/main/inst_deb13_amd64_dev.sh && bash inst_deb13_amd64_dev.sh
```
